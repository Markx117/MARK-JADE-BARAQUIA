Project title: Online Printed T-Shirt Designing

Project Start Date: June 1, 2019

Projected Finish Date: October 31, 2019

Budget Information: 50,000.00

Project Manager: MARK JADE B. BARAQUIA, 09272494865, markjade.baraquia@gmail.com

Project Objectives:

• Online Printed T-Shirt Designing system is a website which you can see some design of the shirts and also can make or customize your own design for you to achieve satisfaction.
• This project aims to provide a user friendly interface so that the user will be use it easily.
• This project enables the user to make their own customizations and own designs.
• Online Printed T-Shirt Designing system aims to achieve success by following its scope and limitation.

Approach:

Online Printed T-shirt Designing will have registration for new Users and login for Members. This software will have Admin Login which will check for new request and orders, Members Login in which they can create customize design using wide range of Colors & Design, and can even place bulk order for the same.
This Ordered T-shirt will be delivered to their door steps. The Admin Login will accept request to check design, approve Order, and create bill.

PROJECT DETAILS

This Project Online Printed T-shirt Design which the user sign up and Log in and create his/her own design and check out , payment method and it will be shipped at his door step. The environment of this project through online transaction. The including language is PHP, Java script, and the hardware are Windows 7, Sql 2008, Visual studio 2010.

This system will be online and can be used by anyone, Just he needs to register, fill a form, Choose Design, Color and select quantity. Then after some days he will have T-Shirt at his door steps. This system is a less hassle for the reason that you are not going to travel far from your home just to find a cheaper printing shop and cheaper T-shirt in order to save money but here all you need is internet to create an order from this system.
